=====
 CSS
=====


Allgemeine Definition einer Regel
	selektor {
		eigenschaft1: wert1;
		eigenschaft2: wert2;
		...
	}

Beispiel mit Kommentaren
	/* Hauptüberschriften in Rot */
	h1 {
		font-family: Verdana;
		font-size: 1.5em;
		color: red;
	}
	/* Links ohne Unterstreichung */
	a {
		text-decoration: none;
	}

------------------
Einbinden von CSS:
------------------

Lokales CSS:

	<p style=“font-family: Arial; color:gray“>
	  Die Schrift ist grau, aber hier ist sie <span style=“color:red“>rot</span>.
	</p>
	<div style=“background-color:gray; font-family: Arial; color:white“>
	  Dieser Kasten ist grau, die Schrift weiß!
	</div>

Zentrales CSS:

	<!doctype html>
	<html>
	<head>
	  <style>
	    body {
	      background-color: white;
	    }
	    p {
	      font-family: Arial;
	      font-size: 0.5em;
	    }
	  </style>
	</head>
	...

Externes CSS (<dateiname>.css):

<link>-Element
	<!doctype html>
	<html>
	<head>
		<link rel=“stylesheet“ href=“styles.css“>
	</head>
	...
@import-Direktive
	<!doctype html>
	<html>
	<head>
	  <style>
	    @import url(“styles.css“);
	  </style>
	</head>
	...

Selektoren:
-----------
			Einfachere schreibweise
			https://code.visualstudio.com/docs/languages/emmet

Einfache Selektoren:

Universalselektor
	* {
		Eigenschaft1: Wert1;
		Eigenschaft2: Wert2;
		...
	}
HTML-Elemente (Tags)
	tag-name {
		Eigenschaft1: Wert1;
		Eigenschaft2: Wert2;
		...
	}
Klassen
	.meine_klasse {
		eigenschaft1: wert1;
		eigenschaft2: wert2;
		...
	}
IDs
	#id {
		eigenschaft1: wert1;
		eigenschaft2: wert2;
		...
}
Attribute
	[Attribut] {
		eigenschaft1: wert1;
		eigenschaft2: wert2;
		...
	}
	[Attribut=Wert] {
		eigenschaft1: wert1;
		eigenschaft2: wert2;
		...
	}

Kombinationen:

Klasse mit HTML-Element
	tag.klasse {
		...
	}
Aufzählung gleichberechtigter Selektoren
	element1, element2 {
		...
	}
element2 innerhalb von element1
	element1 element2 {
		...
	}
element2 als direkte Kinder von element1
	element1 > element2 {
		...
	}
element2 als erstes Kind von element1
	element1 + element2 {
		...
	}
Nesting (mit &)
	.meine_klasse {
		...
		& > tag {
			eigenschaft1: wert1;
			...
		}
	}


------------------------
Pseudo-Klassen/Elemente:
------------------------

Pseudoklassen:

Definieren einen speziellen Zustand eines Elements
	valid/invalid
	required/optional
	visited/unvisited
	...
Werden mit einfachem : abgetrennt

unbesuchter Link
	:link
besuchter Link
	:visited
mit Maus berührter Link (sogenannter „Mouseover“)
	:hover
aktuell angeklickter Link
	:active
ausgewähltes Element, z. B. Formularfeld
	:focus
n-tes Vorkommmen eines HTML-Elements.
	:nth-of-type(n)
	Das n in der Klammer kann eine Ziffer, eine Funktion (z.B. 2n) oder
	ein Schlüsselwort (even für gerade, odd für ungerade) sein.

Pseudoelemente (immer mit ::):

Zielen auf bestimmte Teile eines Elements.
	first letter or line
    content before or after an element
    markers of list items
    viewbox behind a dialog box
Werden mit doppeltem :: abgetrennt

erste Zeile
	::first-line
erster Buchstabe
	::first-letter
Einfügen vor dem Element, content gibt den Inhalt an.
	::before
Einfügen nach dem Element, content gibt den Inhalt an.
	::after
Vom User selektierter Text
	::selection

----------
Vererbung:
----------

CSS-Eigenschaften werden vom Eltern- an die Kind-Elemente vererbt.
Kinder können überschreiben.

Spezifität von Selektoren:

Selektor(kombination)						A	B	C	D	Beispiel
Einfache Selektoren
Universalelement							0	0	0	0	* {...}
HTML5-Element								0	0	0	1	body {...}
Klasse oder Pseudoklasse					0	0	1	0	.rot {...}
Individualformat							0	1	0	0	#blau {...}
Lokale style-Definition						1	0	0	0	<p style=“...“>...</p>
Kombinierte Selektoren
HTML-Element:Pseudoklasse					0	0	1	1	a:hover {...}
HTML-Element HTML-Element					0	0	0	2	article p {...}
Individualformat HTML-Element				0	1	0	1	#blau p {...}
Individualformat HTML-Element:Pseudoklasse	0	1	1	1	#rot a:hover {...}

Höchste Zahl gewinnt!

Rangfolge:

1.	Autoren-CSS	(Vom Entwickler geschrieben)
2.	User-CSS	(User-Einstellungen im Browser)
3.	Browser-CSS	(Browser-Standard)

Maßeinheiten:

Maßeinheiten immer ohne Leerzeichen anhängen!

Absolute Einheiten:

Können nur sinnvoll für Ausdrücke verwendet werden!

Einheit	Bedeutung				Beispiel
px 		Pixel, absolute 		width: 100px;
		Anzahl der Bildpunkte
		eines Endgeräts
mm		Millimeter				height: 20mm;
cm		Zentimeter				margin: 1cm;
pt		Punkt (= 0,3528 mm)		line-height: 12pt;
pc		Pica (= 12 pt)			font-size: 1pc;
in		Inch (= 2,54 cm)		padding: 0.5in

Relative Einheiten:

Einheit	Bedeutung				Beispiel
em		M-Höhe, Bezug:			font-size: 1.2em
		Element-Schriftgröße
rem		M-Höhe, Bezug:			font-size: 2rem
		Root-Schriftgröße
%		Prozent, Bezug zum		width: 50%;
		Elternelement
vw		proz. Anteil an Breite	div: 50vw;
		des Viewports
vh		proz. Anteil an Höhe	margin: 5vh;
		des Viewports

Farbangaben:
------------

Hexadezimale Farben:

#xxxxxx (Rot|Grün|Blau)

#000000	Weiß
#FF0000	Rot
#00FF00	Grün
#0000FF	Blau
#FFFFFF	Schwarz

Dezimale Angaben:

rgb(xxx,xxx,xxx) (Rot|Grün|Blau)
(0-255 oder Prozent)

rgb(0,0,0)			Weiß
rgb(255,0,0)		Rot
rgb(0,255,0)		Grün
rgb(0%,0%,100%)		Blau
rgb(255,255,255)	Schwarz

rgba(xxx,xxx,xxx,y.z)

y.z => Transparenz von 0 (durchsichtig) bis 1.0 (undurchsichtig)

Namen:

Für 140 Farben sind Namen definiert:
	Hexcode	Farbname
	#000000	black
	#808080	gray
	#C0C0C0 silver
	#FFFFFF white
	#00008B darkblue
	#00FFFF cyan
	#006400 darkgreen
	...

Farbverläufe:

Allg. Definition eines linearen Farbverlaufs
	background: linear-gradient(richtung, farbe1, farbe2,...);
	richtung kann z. B. to bottom, to top, to right, to left 
	oder eine Winkelangabe z. B. 45deg sein.
Allg. Definition eines radialen Farbverlaufs
	background: radial-gradient(farbe1, farbe2,...);
Beispiel
	<!doctype html>
	<html>
	<head>
		<style>
			#box1 {
				height: 200px;
				width: 200px;
				background: linear-gradient (45deg,yellow,red);
			}
			#box2 {
				height: 200px;
				width: 200px;
				background: radial-gradient (cyan,blue);
			}
		</style>
	</head>
	<body>
		<div id=“box1“></div>
		<div id=“box2“></div>
	</body>
	</html>

Schriftarten:
-------------

Verwendung einer Systemschrift
	font-family: Arial;
	font-family: “Times New Roman“;

Einbinden einer Schrift mit @font-face
	@font-face {
		font-family: Roboto;
		src: url(Roboto-Black.ttf);
	}
Beispiel
	<style>
		@font-face {
			font-family: Roboto;
			src: url(Roboto-Black.ttf);
		}
		p {
			font-family: Roboto, Arial, sans-serif;
			font-size: 2.0em;
		}
	</style>

Flexbox:
--------

Flex-Container:

Erzeugen eines Flex-Containers
	display: flex;
Anordnung der Flex-Items
	flex-direction: row;			nebeneinander
	flex-direction: row-reverse;	nebeneinander rückwärts
	flex-direction: column;			untereinander
	flex-direction: column-reverse;	untereinander rückwärts
Horizontale Anordnung der Flex-Items
	justify-content: flex-start;	links nach rechts
	justify-content: center;		zentriert
	justify-content: flex-end;		rechts nach links
	justify-content: space-between;	Leerraum dazwischen
	justify-content: space-around;	Leerraum dazwischen und am Rand
Vertikale Anordnung der Flex-Items
	align-items: flex-start;	oben
	align-items: center;		mittig
	align-items: flex-end;		unten
	align-items: baseline;		Grundlinie des Containers
	align-items: stretch;		strecken um auszufüllen
Umbruch der Elemente
	flex-wrap: nowrap;			Alle Elemente in einer Zeile
	flex-wrap: wrap;			Umbrechen auf zusätzlich Zeilen
	flex-wrap: wrap-reverse;	Umbrechen in umgekehrter Reihenfolge
Abstand zwischen mehreren Zeilen
	align-content: flex-start;		oben
	align-content: flex-end;		unten
	align-content: center;			mittig
	align-content: space-between;	Leerraum dazwischen
	align-content: space-around;	Leerraum dazwischen und am Rand
	align-content: stretch;			strecken


Flex-Items:

Reihenfolge
	order: n;
	Hinweis: n ist eine ganze Zahl, 1 ganz oben bzw. ganz links
Ausdehnungsverhalten
	flex-grow: n;
	Hinweis: n ist eine ganze Zahl, 0: unveränderlich, 1: leicht veränderlich, 2: stärker veränderlich usw.
